# poneglyph-generator
A static web page that generates One Piece poneglyph-like text based on user input

This code uses a custom font called Poneglyph.ttf shared by [Killiam Frost at Reddit](https://www.reddit.com/r/OnePiece/comments/9hyc1g/poneglyph_font_ttf_file/).

Available at: https://pitossomos.github.io/poneglyph-generator/
